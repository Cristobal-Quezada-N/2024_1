# Descargar Archivos de Replit

Descargar archivos de un team de Replit (se requiere de un usuario integrante del grupo).

## Manual de Uso

### Install

Instalar librerias e inicializar entorno virtual.

```bash
make install
```

### Run

Ejecutar script.

```bash
make run
```

### Clean

Borrar el entorno virtual.

```bash
make clean
```
